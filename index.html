我是國立台北教育大學資訊科學系二年級的學生，我非常熱愛排球，並且積極參與系隊活動，不僅提升了自己的體能，也增進了與隊友間的默契與合作能力。在大學生活中，我學習如何使用VScode 撰寫 HTML 和 Java 程式，並且在大一時已掌握撰寫中階 C++程式的能力。此外，我對於程式設計的熱情驅使我不斷探索新的技術與工具，讓自己能在資訊領域持續進步與成長。
